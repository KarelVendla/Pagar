# Pagar
Lävendi töö

Tuleb teha veebileht, millel hoitakse vähemalt kahe omavahel seotud pärismaailma objekti andmeid:

1.       Kummalegi andmetüübile luua CRUD vormid

2.       Kujundada veebileht ühtse kujundusega, mis erineb vaikimisi kujundusest (bootstrap)

3.       Lisada 2 rolli (administraator on parooliga kaitstud)

4.       Lisada leht, kus kuvatakse baasi salvestatud andmete aruannet (agregaat funktsioonid)

5.       Alamprogrammidele ja põhi konstruktsioonidele lisada dokumenteeritavad kommentaarid

6.       Dokumentatsioon ja versioonihaldus Github-is
